Centimeters to Inches Converter

Problem: Write a program that converts centimeters to inches.

Pseudocode

Ask user for number
Inches = (Centimeters) * .393701;

Solution

var centimeters = parseInt(document.getElementById('centimeters').value);
var inches = (centimeters) * .393701;
document.getElementById("inches").innerHTML = "Inches = " + inches;

Random Number Generator

Problem: Write a program that generates a random number through 100.

Pseudocode

Click the "Generate Number" button ro generate a random number between 1 and 100.

Solution

var x = document.getElementById("demo")
x.innerHTML = Math.floor((Math.random() * 100) + 1);
